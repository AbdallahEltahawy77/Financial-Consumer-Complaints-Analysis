# Customer Complaints Intelligence Dashboard 📊

## 📌 Project Overview
An end-to-end Power BI project analyzing financial consumer complaints. This dashboard transforms raw data into actionable insights using advanced DAX, custom UI design, and interactive features.

## 🚀 Key Features
- *Professional UI/UX:* Custom-designed background using Canva for a modern Dark Mode experience.
- *Advanced DAX:* Complex measures for Timely Response Rates and Daily Averages.
- *Dynamic Tooltips:* Innovative Gauge Charts (Speedometer style) for instant product performance checks.
- *Geographical Analysis:* Map integration to identify complaint hotspots across US states.
- *Drill-through Capability:* Detailed view for individual complaint records.

## 🛠️ Tools Used
- *Power BI:* Data Visualization & Modeling.
- *Power Query:* Data Cleaning & Transformation.
- *DAX:* Statistical Analysis.
- *Canva:* UI/UX Design.
